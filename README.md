## Quick start

### Generate Gemini API

Access [aistudio.google](https://aistudio.google.com/) create `GEMINI_API_KEY`

```json
NEXT_PUBLIC_GEMINI_API_KEY=AIza******msw
```

Run:

```bash
yarn dev
```
